# ESW 2025-2026 Projects

Information about the projects for the ESW 2025-2026 academic year can be found in
[this Google Doc](https://docs.google.com/document/d/1Cw7M3cpmoSit890-Cm11XMuY_vEbCZrrylU3aPUu5oU/edit?usp=sharing).

It is important to note that currently, the firmware infrastructure provided by this repository
only supports the STM32G4 series. Other families of MCUs can be supported, but the associated
firmware and HAL libraries must be added to `lib/` and changes must be made throughout the
stack to use the correct drivers.
